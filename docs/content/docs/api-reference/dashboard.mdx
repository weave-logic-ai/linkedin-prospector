# Dashboard API

A single endpoint that returns all dashboard data in one request, including contact
statistics, tier distribution, enrichment budget, network health, goals, tasks, and
recent discoveries.

---

## Get Dashboard Data

```
GET /api/dashboard
```

Returns aggregated data for the main dashboard. Runs multiple parallel database queries
for optimal performance.

**Auth:** None

### Response Fields

**stats** — Contact overview:

| Field | Type | Description |
|-------|------|-------------|
| `totalContacts` | number | Non-archived contact count |
| `tierDistribution` | object | Count per tier (gold, silver, bronze, watch, unscored) |
| `enrichedCount` | number | Total enrichment records |
| `enrichmentRate` | number | Ratio of enriched to total contacts |
| `recentImports` | number | Import sessions in last 7 days |

**budget** — Enrichment budget period:

| Field | Type | Description |
|-------|------|-------------|
| `budgetCents` | number | Budget for current period |
| `spentCents` | number | Amount spent |
| `utilization` | number | Spent / budget ratio |
| `lookupCount` | number | Total lookups this period |

**networkHealth** — Data quality indicators:

| Field | Type | Description |
|-------|------|-------------|
| `dataMaturity` | number | 0-100 score (contacts + enrichment + graph + embeddings) |
| `graphMetricsComputed` | boolean | Whether PageRank etc. have been computed |
| `embeddingsGenerated` | number | Profile embeddings count |
| `totalEdges` | number | Network edges count |
| `avgScore` | number | Average composite score |

**activeGoal** — Highest-priority active goal (or `null`).

**pendingTasks** — Top 5 pending tasks by priority. Priority labels: `high` (1-3),
`medium` (4-6), `low` (7-10).

**recentDiscoveries** — Latest 5 niches and ICPs, sorted by creation date.

**curl:**

```bash
curl http://localhost:3750/api/dashboard
```

**Error Codes:** `500`
